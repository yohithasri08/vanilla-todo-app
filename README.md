# TaskFlow Dashboard

## 📸 Screenshots

### Home Dashboard

TaskFlow Dashboard is a premium productivity application built with pure HTML, CSS, and Vanilla JavaScript. It combines thoughtful UX, polished state management, and modern visual design to deliver a portfolio-ready task manager that feels like a real SaaS product.

## Features

- Create, edit, delete, and complete tasks
- Organize tasks with priorities, categories, and due dates
- Search and filter tasks in real time
- Sort tasks by newest, oldest, name, priority, and due date
- Monitor progress through animated dashboard cards and a polished progress bar
- Switch between light and dark mode with saved preference
- Persist data in localStorage for reliable session continuity
- Enjoy modern toast notifications and a refined modal editing experience

## Architecture Overview

The app is structured around a single-page experience with:

- A stateful task array that powers CRUD behavior
- Centralized rendering for the task list and dashboard
- Event delegation for task interactions
- Local persistence for durable state management
- Modular UI helper functions for filtering, sorting, and rendering

## Installation

No installation is required.

1. Open the project folder.
2. Launch [index.html](index.html) directly in your browser, or run a local server:

```bash
python -m http.server 8000
```

3. Visit http://localhost:8000 in your browser.
